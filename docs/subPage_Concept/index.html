Concept

Rosalieでは、季節ごとに旬のブーケを多数取り揃えています。
その時期に最も美しい花々を厳選し、丁寧にアレンジをしています。
日常の生活を細やかに彩るための、こだわりの花を是非お楽しみください。

Rosalieの3つのこだわり

デイリー向け商品の豊富なラインナップ

毎日の生活に花を取り入れやすいよう、デイリー向け商品を豊富に取り揃えています。
手軽に楽しめる小さなブーケや、お部屋を華やかにするアレンジメントまで、日々の生活に彩りを添える花々が揃っています。
お仕事帰りやちょっとしたお買い物のついでに立ち寄っていただけるよう、忙しい日常に癒しの瞬間をお届けします。

パリ風の洗練されたフラワーデザイン

当店のフラワーデザインは、パリのマルシェにインスパイアされたエレガントでナチュラルなスタイルです。
季節感あふれる花々を使ったデザインは、まるでパリの街角にいるかのような特別な気分を味わえます。
日常をちょっとおしゃれに、そして特別にする花々をお楽しみください。

鮮度にこだわった新鮮な花々

花の鮮度と品質に強いこだわりを持ち、信頼できる産地から新鮮な花々を仕入れています。
また、お客様に新鮮で美しい花をお届けできるよう、厳しい品質管理を行っています。
花の栄養剤やケア用品もご用意しておりますので、是非ご活用ください。

Rosalieの「Flower」ページでは、鮮度とデザインにこだわった様々な花をご覧いただけます。
デイリーにぴったりなアイテムや特別なデザインのブーケを取り揃えています。
